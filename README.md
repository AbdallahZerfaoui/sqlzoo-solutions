# SELECT basics
We are going to use a table called World for the next exercices.
#### 1. show the population of Germany
```SQL
  SELECT population 
  FROM world
    WHERE name = 'Germany'
```
#### 2. show the population of Germany
```SQL
  SELECT population 
  FROM world
    WHERE name = 'Germany'
```
